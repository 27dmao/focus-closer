Welcome to Focus Closer.

This extension closes distracting tabs before you can scroll them. YouTube videos get a Claude-powered content verdict. Domains on your blocklist instantly close. False positives recover in 5 seconds.

To classify YouTube, it needs an Anthropic API key. ~$0.30/month at typical use.

Step 2 of 3

Pick your strictness.

Strict = ambiguous defaults to close. Balanced = ambiguous defaults to keep. Both have the 5-second recovery popup either way.

Step 3 of 3

You're set.

Try it now:

  1. Visit a YouTube video you know is distracting — it should close within a second.
  2. On any blocklist domain (instagram.com, x.com, linkedin.com/feed), the tab closes instantly.
  3. Press Ctrl+Shift+X to flag a video the classifier missed.
  4. Press Ctrl+Shift+P to pause the extension for an hour.

Start a focus session

Declare what you're working on. During the session, the classifier runs stricter — borderline "productive" YouTube videos default to close instead of keep. Cache is bypassed so everything re-classifies against the tighter bar.

How it differs from always-on mode

  • Strict boost. Low-confidence "productive" verdicts default to close during a session. Normal mode keeps them.
  • Cache bypass. Every video re-classifies fresh — so a previously-productive video gets the tighter bar applied too.
  • Completion ritual. End-of-session toast tells you how many distractions got blocked — small dopamine reward for finishing.
  • Log tag. Closes during a session are tagged so the Insights tab can show session-vs-ambient patterns.

How classification works

  1. User override — videos you've manually flagged always close; videos you've reopened as false positives always stay.
  2. Cache — videos classified within the last 30 days use the cached verdict instantly.
  3. Local rules — channel whitelist / blocklist and keyword matching on title/description. Free, ~0 latency. Covers ~50% of cases.
  4. Claude Haiku 4.5 — remaining ambiguous cases. ~$0.0005/call, ~500ms. Returns productive / unproductive with reason.
  5. Fail-open — if Claude is unreachable, the tab stays. Never silently break browsing.

Commitment device

The extension alone is bypassable — Guest mode, Incognito, or a fresh Chrome profile all defeat it. For real enforcement, pair with a Chrome managed policy that disables Guest / Incognito / new profiles. On macOS that lives at /Library/Managed Preferences/com.google.Chrome.plist and requires sudo to install or remove. See the README for the policy file.

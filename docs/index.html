Instagram and X are always distractions — a domain block is fine. LinkedIn is mostly work. YouTube is the hard one: the same URL pattern (/watch) holds both Khan Academy and Minecraft. A blunt block is wrong. That's where an LLM earns its keep.

Two subsystems. One 5-second recovery loop.

LLM-classifying every tab is expensive, slow, and risks misfiring on your Gmail. So Focus Closer has two engines — a cheap blocklist handles obvious distractions; Claude handles only the ambiguous content at youtube.com/watch.

The four design calls that matter

01 — Strictness

Close first, recover after.

Asking "are you sure?" before every close is exhausting and trains you to dismiss the modal. Close immediately with a visible Reopen button — only the actual false positives cost you 2 seconds.

02 — Architecture

Not every tab needs an LLM.

Two independent subsystems: an instant-close blocklist for always-distracting sites, and a Claude classifier for the one place LLM reasoning earns its keep — YouTube's mixed productive/unproductive stream.

03 — Granularity

Path-level, not domain-level.

linkedin.com/feed is a doomscroll surface. linkedin.com/messaging is how leads reach out. The blocklist distinguishes them, with work-whitelist hardcoded for Gmail, Calendar, Claude, Apollo, Symbal, and the rest.

04 — Learning

The classifier gets smarter each week.

Every popup offers "Always allow this channel" / "Always block this channel" — one click teaches the system forever. AI weekly insights analyze your log and surface patterns you wouldn't notice on your own.

Features

★

Claude classifier

Reads title, channel, description, tags. Returns productive/unproductive with reason. ~500ms, ~$0.0005 per call.

⏻

Focus Sessions

Timer-based deep work mode. 25/50/90-min presets. During a session, the classifier runs stricter and cache bypasses. End-of-session toast tells you how many distractions got blocked.

✦

AI Insights

Claude reads your last 7 days of closes and writes a personalized brief: pattern observed, biggest attention leak, one thing to try. Nothing else in this space does this.

↻

5-second recovery

False positives aren't punishing. Reopen button restores the tab and teaches the override. ESC to dismiss. Hover to pause timer.

☐

Attention score

Single number, trends over time. Behavior-change products work because progress is legible.

⌘

Keyboard-native

⌘+Shift+X marks distracting. ⌘+Shift+S marks productive. Each press updates the channel lists, so the classifier converges fast.

⌇

Path-level blocklist

linkedin.com/feed blocked, linkedin.com/messaging untouched. Granularity without fiddling.

◎

Commitment device

Pairs with a macOS Chrome managed policy that disables Guest mode, Incognito, and new-profile creation — requires sudo to install and remove.

Install

Five minutes. No build step, no dependencies, no server. Pure vanilla JS + Manifest V3.

1
Clone the repo
git clone https://github.com/27dmao/focus-closer
2
Load unpacked in Chrome
Open chrome://extensions → turn on Developer mode → Load unpacked → select the repo folder.
3
Onboarding will prompt you for your Anthropic API key. Get one at console.anthropic.com. Budget $5 — that's over a year at your rate.
4
Optional but recommended: commitment device. See the README for the macOS Chrome managed policy that disables Guest/Incognito/new-profile creation. Requires sudo.
